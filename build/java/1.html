What is Java?

Java is one of the most widely used programming languages.A programming language is used to command a computer to do a variety of tasks. You may have heard of Java because of the popular game Minecraft. This is ironic, though, because Java is not intended for games at all.


A basic Java program looks like this:

public class HelloWorld {

    public static void main(String[] args) {
        // Prints "Hello, World" to the terminal window.
        System.out.println("Hello, World");
    }

}
