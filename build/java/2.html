Installation and Your First Program


What do we use to program Java?

In order to compile and run Java, the Java Development Kit (JDK) is required. In its most basic definition, compiling a program is to take text written by humans and turn it into something a computer can understand. To make life a whole lot easier, we will be using what is called an Integrated Development Environment, or "IDE" for short. An IDE lets us write, compile, and run code all in one place. In this class, we will be using the Eclipse IDE.


Installing the JDK

In order to begin programming, we first have to install the Java Development Kit. You can get the JDK from Oracle's website, here.

  1. Click the download button that has the Java logo and says Java Platform (JDK) 8u144 (the version number may be different).
  2. Click Accept License Agreement under the first box.
  3. Select the correct version to download. For most, the Windows x64 version will be correct.
  4. Run the downloaded program to begin installation. For most users, just clicking "Next" when prompted is okay.
  5. On Windows, check that the JDK was successfully installed by looking in C:\Program Files\Java. There should be two folders, with names containing "jdk" and a "jre".

Installing Eclipse

The next step in getting ready to program is to install an IDE. We will be using the Eclipse IDE. Eclipse is one of the most popular Java IDEs, alongside IntelliJ Idea and Netbeans. As mentioned above, using an IDE is leaps and bounds better than hand writing and compiling code. IDEs typically come with nifty and useful features such as code completion and error detection. Eclipse is available for download from the Eclipse Foundation website, here.

  1. For most Windows users, the Windows 64-bit download will be correct. There are also OSX and Linux versions available.
  2. The downloaded .zip file will have all you need to begin using Eclipse. There is no installation needed.
  3. Extract the contents of the .zip folder (the whole folder named eclipse, not just the stuff inside it) into a convinient location on your computer. I recommend the Documents folder.
  4. Run eclipse.exe.
  5. Upon launch, Eclipse will prompt for a workspace location. It is recommended to keep this as the default.
  6. When Eclipse opens up to the "Welcome" page, you are ready to create your first project.

Creating Your First Eclipse project

Now we can finally create our first program! In order to begin, we need to create a new project:

  1. Navigate to File > New > Java Project.
  2. Name the project Hello World.
  3. Leave the JRE settings the same, but if you have problems executing your program, try selecting "Use default JRE".
  4. Click Finish.

We now have our Eclipse Java project. In the package explorer to the right, you can see the project folder, and can click the small arrow to view the contents of the project. Inside the project folder, there should be JRE System Library and src. The src folder is where all of our code goes. Right now, it's empty. Let's fix that.

  1. Right-click the project and select New > Package.
  2. The typical naming convention is to prefix the package name with a backwards domain name. For example, a package created by someone at Google could be com.google.packagename. Additionally, package names should only contain numbers and lowercase letters, with no spaces.
  3. Since it is unlikely that you have a domain name, let's just use me.yourname for now. For example, me.mitchell.helloworld.

Now that we have a package, let's create a class and add some code:

  1. Right-click our newly created package and navigate to New > Class.
  2. Name the class HelloWorld. There is a naming convention with classes, too. Generally, they are written in UpperCase style, with the first letter of every word being capitalized. Names also must not contain spaces.

After clicking Finish, you should have something that looks like this:

package me.mitchell.helloworld;

public class HelloWorld {

}

Let's add some code to this and run it. Don't worry, I'll explain what all this does in the next section.

Paste this in between the curly braces (the { and }):

public static void main(String[] args) {
    // Prints "Hello, World" to the terminal window.
    System.out.println("Hello, World");
}

Your HelloWorld class should now look something like this:

package me.mitchell.helloworld;

public class HelloWorld {
    public static void main(String[] args) {
        // Prints "Hello, World" to the terminal window.
        System.out.println("Hello, World");
    }
}

Now run the file by clicking the green arrow button at the top right. This is the run button. The text "Hello, World" should be displayed at the bottom of your screen.

Pretty neat, right? Okay, maybe not that cool, but it will get better soon. In the next section, I'll go through this line-by-line and explain how a basic Java program works.
